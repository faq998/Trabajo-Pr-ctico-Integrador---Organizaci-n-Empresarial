# Trabajo Práctico Integrador - Organización Empresarial

## Descripción

Este proyecto tiene como objetivo simular la automatización del proceso de nominación de Gas Natural utilizado en el ámbito de la generación eléctrica.

Mediante un chatbot desarrollado en Python, se representa la interacción entre EPEC, el proveedor de Gas Natural (Trafigura) y CAMMESA, permitiendo gestionar solicitudes de combustible, analizar la disponibilidad y determinar la forma de operación de las centrales de generación.

## Funcionalidades

* Solicitud de Gas Natural.
* Validación de cantidades ingresadas.
* Simulación de respuesta del proveedor.
* Declaración de cupo a CAMMESA.
* Determinación del combustible de operación.
* Gestión de estados mediante Máquina de Estados Finita (FSM).
* Manejo de errores y situaciones excepcionales (camino infeliz).

## Tecnologías utilizadas

* Python
* Visual Studio Code
* Excel (Base de datos simulada)
* BPMN 2.0 (Bizagi Modeler)
* Git y GitHub

## Estructura del proyecto

* `Chatbot.py`: programa principal.
* `Datos complementarios.xlsx`: base de datos simulada y diccionario de datos.
* `Modelo BPMN/`: diagramas del proceso.
* `Interacciones chatbot`: captura de las interacciones con el chatbot.
* `README.md`: documentación del proyecto.

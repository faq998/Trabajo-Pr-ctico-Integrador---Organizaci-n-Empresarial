# Trabajo Práctico Integrador - Organización Empresarial

## Descripción

Automatización del proceso de nominación de Gas Natural mediante un chatbot simulado.

## Tecnologías utilizadas

- Python
- Visual Studio Code
- Excel (Base de datos simulada)
- BPMN 2.0 (Bizagi Modeler)

## Ejecución

1. Abrir el archivo chatbot_gn.py
2. Ejecutar el programa desde Visual Studio Code.
3. Seguir las instrucciones mostradas en pantalla.
